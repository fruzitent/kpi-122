\begin{abstract}
    \section*{Мета роботи}
    \label{sec:meta}

    \begin{enumerate}
        \large

        \item
        Навчитись ранжувати альтернативи за числовими та якісними характеристиками

        \item
        Навчитись будувати граф бінарних відношень

        \item
        Оволодіти навичками ранжування альтернатив за методом рядкових сум
    \end{enumerate}


    \section*{Завдання}
    \label{sec:tasks}

    \begin{enumerate}
        \large

        \item
        Навести приклад реальної задачі прийняття рішень,
        в основу якої покладено ранжування 5 конкретних альтернатив

        \item
        На основі наведених даних побудувати граф бінарних відношень
        для альтернатив за будь-яким одним з атрибутів

        \item
        На основі візуального аналізу побудованого графа,
        перевірити чи не порушена властивість транзитивності бінарних відношень.
        Сформулювати означення транзитивності та навести приклад графу
        для будь-яких альтернатив, де порушена транзитивність бінарних відношень

        \item
        Навести приклад таблиці попарних порівнянь 5 конкретних альтернатив
        з непорушеною властивістю транзитивності та провести
        їх ранжування за методом рядкових сум.
    \end{enumerate}
\end{abstract}
